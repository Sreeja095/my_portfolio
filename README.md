## 🚀 About

A modern, responsive personal portfolio website built to showcase my journey as a tech enthusiast — featuring my projects, technical skills, achievements, and interests. The site is designed with a premium glassmorphism aesthetic, smooth animations, and a fully responsive layout that works beautifully across all devices.

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🌗 **Dark / Light Theme** | Seamless toggle between dark and light modes with the theme preference saved for return visits |
| ⌨️ **Typing Animation** | A dynamic typing effect in the hero section that cycles through different roles |
| 🎨 **Glassmorphism UI** | Frosted-glass cards with soft blur and transparency for a clean, premium look |
| 🔢 **Animated Counters** | Stats like certificates, projects, and technologies count up as you scroll into view |
| 📊 **Skill Progress Bars** | Visual progress bars that animate to show proficiency levels across different skill categories |
| 📱 **Fully Responsive** | Mobile-first design with a hamburger navigation menu and fluid grid layouts |
| 🌀 **Scroll Reveal Effects** | Sections and cards gracefully fade and slide in as the user scrolls down the page |
| 🎭 **Animated Background** | Floating gradient orbs and a subtle grid overlay that bring the background to life |
| 📧 **Contact Form** | Built-in form with email, subject, and message fields to get in touch directly |
| ⬆️ **Back to Top** | A floating button that appears on scroll for quick navigation back to the top |
| ⚡ **Preloader** | A smooth loading spinner that transitions out once the site is ready |

---

## 🧭 System Flow

1. **Preloader** — A loading spinner is shown while assets load, then fades out smoothly
2. **Hero Section** — Greeting with a typing animation, call-to-action buttons for Resume and Contact, and social links
3. **About Me** — A short bio along with animated stat counters highlighting 17+ certificates, 13+ projects, and 9+ technologies
4. **Technical Skills** — Four categories (Frontend, Backend, Data Analysis, Tools & Practices) displayed with animated progress bars
5. **Featured Projects** — Showcase of key projects including Weather App, Tic-Tac-Toe, and To-Do List, each with live demo and source links
6. **Achievements** — Highlights including Square Hackathon Finalist at IIT Delhi, GreenHerbaCureDB, MITTI smart agriculture, and IoT Smart Garbage Collection
7. **Hobbies & Interests** — Photography, Music, Traveling, Reading, and Art & Design
8. **Contact** — A contact form alongside email, LinkedIn, GitHub links and location (Kolkata, India)
9. **Footer** — Copyright and social icons

The site is deployed on **Netlify** with continuous deployment from GitHub — every push to the main branch automatically updates the live site. No build step is required.
